Cenário atual
📋

Planilhas desatualizadas

Frequência e escala controladas em Excel, sem histórico confiável, sujeitas a erros humanos e versões duplicadas.

📦

Estoque no papel

Materiais vencendo sem alerta, produtos zerados sem que a gestão saiba, movimentações sem rastreabilidade.

🚌

TFD sem controle

Encaminhamentos de pacientes anotados manualmente, sem padronização de documento nem histórico centralizado.

🏛️

Relatórios para o Ministério da Saúde

Geração manual do BPA para o DATASUS, processo lento e propenso a rejeições por inconsistência.

A solução existe — e foi construída

O SMS Conceição resolve todos esses pontos em um sistema web moderno, acessível de qualquer dispositivo, sem instalação, com dados na nuvem e relatórios prontos para impressão.

Módulo Frequência em detalhe

Do espelho individual à geração em lote — em segundos

Como funciona
1

Configurar o mês

Selecione mês, ano, tipo de período e informe dias facultativos específicos do município.

2

Gerar em lote

Um clique gera os espelhos de todos os servidores — diferenciando escala rotativa de regime padrão.

3

Editar individualmente

Ajuste faltas, horas extras ou ocorrências específicas de cada servidor antes de imprimir.

4

Imprimir com um clique

Documento formatado, pronto para assinatura, no padrão exigido pelo TCE/TCM.

Diferenciais

🗓️ Feriados automáticos

Feriados nacionais e municipais já marcados automaticamente na grade de frequência.

📋 Escala diferenciada

Servidores em escala recebem folha em branco; demais recebem grade Seg–Sex preenchida.

🔒 Transparência pública

Exportação de dados de frequência para portal de transparência, com sincronização automática.

🏥 CNPJ e cabeçalho da SMS

Documento impresso já com os dados institucionais da secretaria — sem configuração manual.

Tecnologia & Segurança

Infraestrutura de nível enterprise, custo acessível

Stack tecnológico
Next.js 15 + React Frontend
API Routes Node.js Backend
Supabase · PostgreSQL Banco de dados
Vercel · CDN global Deploy / Hospedagem
pgcrypto · bcrypt Segurança
Garantias de segurança

🔐 Senhas criptografadas

Nenhuma senha é armazenada em texto puro — bcrypt com salt no banco de dados.

🌐 HTTPS obrigatório

Toda comunicação criptografada via TLS/SSL — padrão em todas as conexões.

🛡️ Credenciais protegidas

Chaves do banco de dados nunca expostas ao navegador — processamento exclusivo no servidor.

💾 Backup automático

Supabase mantém backups diários automáticos com retenção de 7 dias no plano gratuito.
